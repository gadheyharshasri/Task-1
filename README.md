# Task-1
# Network Reconnaissance using Nmap

## Objective

The objective of this task was to perform basic network reconnaissance using Nmap to identify active hosts, open ports, and running services within a local network.

## Tools Used

* Nmap
* Wireshark (optional)

## Procedure

1. Installed Nmap and configured the environment.
2. Identified the local network IP range.
3. Performed a TCP SYN scan using Nmap.
4. Detected active hosts and open ports.
5. Analyzed common services running on discovered ports.
6. Observed packet traffic using Wireshark.
7. Saved scan results in a text file.

The scan successfully identified active devices and exposed network services such as HTTP, HTTPS, SMB, and MySQL. This task helped in understanding basic network scanning, service enumeration, and security risk identification.
